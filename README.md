# 🕌 Masjid Web

Website masjid berbasis Laravel yang memiliki fitur:

## ✨ Fitur
- Jadwal sholat otomatis (API)
- CRUD kegiatan (admin)
- Galeri kegiatan
- Donasi
- Kontak

## 🛠️ Tech Stack
- Laravel 11
- Tailwind CSS
- MySQL
